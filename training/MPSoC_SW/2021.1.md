# Lab 11

## 3-6-4

**Error**

Need correct sdhci controller, in other case, reboot will be cause boot error.

**Fix**

```
&watchdog0 {
	status = "okay";
	timeout = <60>;
	reset-on-timeout;
};

&sdhci1 {
	status = "okay";
	xlnx,has-cd = <0x1>;
	xlnx,has-power = <0x0>;
	xlnx,has-wp = <0x1>;
	disable-wp;
	no-1-8-v;
};
```
